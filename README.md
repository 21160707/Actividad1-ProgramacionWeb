# Portafolio Completo de Prácticas - Programación Web
### Instituto Tecnológico de Oaxaca
**Ingeniería en Sistemas Computacionales**

Este repositorio contiene la colección completa de ejercicios desarrollados para la materia de **Programación Web**. El portafolio abarca la progresión del aprendizaje desde los conceptos fundamentales de marcado semántico de HTML5, pasando por la maquetación responsiva y estilizada con CSS3, hasta el procesamiento dinámico y validación interactiva mediante JavaScript.

---

## 📂 Estructura y Descripción de las Actividades

### 1. Fundamentos de Estructura y Contenido (HTML5 Semántico)
* **Ejercicio 1: Hola Mundo:** Configuración del entorno de desarrollo local con XAMPP (Apache) y estructuración mínima indispensable de un documento HTML5.
* **Ejercicio 2: Párrafos y Formato Básico:** Implementación correcta de etiquetas de texto estructurales y formateo semántico (`<p>`, `<strong>`, `<em>`, y `<br>`).
* **Ejercicio 3: Enlaces Simples:** Manejo de hipervínculos internos de navegación local y externos utilizando los comportamientos de ventana `target="_blank"` y `target="_self"`.
* **Ejercicio 4: Imágenes Básicas:** Inserción adaptativa de recursos gráficos empleando la etiqueta autocerrada `<img>` junto a atributos esenciales de accesibilidad y SEO (`src`, `alt`, `width`).
* **Ejercicio 5: Combinando Texto y Enlaces:** Integración avanzada de elementos de texto, imágenes anidadas dentro de bloques de párrafo y enlaces absolutos con propósitos descriptivos.
* **Ejercicio 6: Secciones Semánticas Básicas:** Organización de la arquitectura de información de una página web empleando los contenedores de bloque modernos de HTML5 (`<header>`, `<section>`, `<footer>`).
* **Ejercicio 7: Listas Anidadas + Enlaces:** Representación jerárquica de datos mixtos mediante la correcta anidación de listas ordenadas (`<ol>`) y desordenadas (`<ul>`) para estructurar recetas o menús complejos.

### 2. Componentes Avanzados y Maquetación Estilizada (CSS3)
* **Ejercicio 8: Horario de Clases:** Creación de estructuras tabulares complejas empleando celdas combinadas mediante los atributos de alcance `rowspan` y `colspan`. El diseño visual se gestiona mediante CSS aplicando selectores estructurales avanzados (`:nth-child(even)`).
* **Ejercicio 9: Formulario de Contacto Moderno:** Interfaz estilizada de captura de datos que integra el uso avanzado de **Variables CSS (`:root`)** para la centralización cromática, reseteo de caja general y maquetación en rejilla bidimensional a través de **CSS Grid**.

### 3. Lógica Interactiva del Lado del Cliente (JavaScript ES6)
* **Ejercicio 10: Calculadora Conversora de Temperatura:** Aplicación funcional e interactiva en tiempo real que captura las entradas del usuario correspondientes a grados Celsius y procesa de forma matemática la conversión al sistema Fahrenheit. Implementa validaciones condicionales lógicas (`isNaN`) para evitar campos vacíos o cadenas incompatibles y utiliza la propiedad protectora `readonly` para impedir manipulaciones manuales del producto de salida.

---

## 🛠️ Tecnologías Empleadas
* **HTML5 Nactivo** (Estructura jerárquica y semántica).
* **CSS3 Custom Properties / Grid Layout / Flexbox** (Estilos globales e interfaces responsivas).
* **JavaScript síncrono** (Manipulación avanzada del DOM y lógica del cliente).
* **Google Fonts API** (Integración tipográfica externa).
